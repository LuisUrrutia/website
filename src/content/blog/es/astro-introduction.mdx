Al reconstruir mi portfolio, evalué muchos frameworks. Astro destacó por varias razones.

## Cero JavaScript por Defecto

Astro envía cero JavaScript a menos que lo necesites explícitamente. Esto resulta en cargas de página increíblemente rápidas.

```astro
---
// Esto se ejecuta en tiempo de build
const posts = await getCollection("blog");
---

<ul>
	{posts.map((post) => <li>{post.data.title}</li>)}
</ul>
```

## Arquitectura de Islas

Agrega interactividad solo donde se necesita con directivas de cliente.

```astro
<!-- Solo carga JS para este componente -->
<ReactCounter client:visible />

<!-- Estático, sin JS -->
<StaticHeader />
```

## Content Collections

Gestión de contenido con seguridad de tipos integrada.

```typescript
const blogCollection = defineCollection({
	schema: z.object({
		title: z.string(),
		date: z.date(),
		tags: z.array(z.string()),
	}),
});
```
